# Flash-VAReason

**Preserving Acoustic Cues for Video Reasoning: An Efficient Uniqueness-Driven Token Compression Framework**

Flash-VAReason is a visual-audio guided video reasoning framework that preserves informative acoustic cues while compressing audio tokens efficiently for MLLM-based reasoning.
